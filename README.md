# Student Performance Prediction

## Overview
This project predicts student final grades using Machine Learning and Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Dataset
UCI Student Performance Dataset

## Features Used
- Study Time
- Failures
- Absences
- G1
- G2

## Model
Linear Regression

## Results
- R² Score: 0.78
- Mean Squared Error (MSE): 4.47

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Prediction
8. Model Saving with Pickle
